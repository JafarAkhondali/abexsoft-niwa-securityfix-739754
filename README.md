# Niwa

Niwa will be a web application library to make 3D worlds and to access the worlds by browser easily. 

## Usage

   $ git clone https://github.com/abexsoft/niwa.git  

## Structure

   * src/           : niwa sources.   
   * libs/          : external libraries, which are three.js and cannon.js.
   * apps/          : examples.
